## About this repository
Tech Stack used:
- React JS (Typescript)
- Tailwind CSS (Daisy UI)

## How to run project
- Clone or make a fork of this project
- Pull to your local repository
- install dependency from package.json using `npm install`
- Run project using `npm start`
